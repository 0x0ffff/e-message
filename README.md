# Message

Vue.js message plugin for vue 3 (from element-plus)


## Import

```js
import Message from "src/index";

createApp(App).use(Message).mount("#app");
```

## Usage

```js
this.$message({ message: 'Hey! I'm here', type: 'success'});
this.$message.success(`Hey! I'm here`);
```
